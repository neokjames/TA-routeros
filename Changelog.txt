## v1.0.0

Forked from https://github.com/schose/TA-routeros.

- Removed RouterOS API modular inputs (routerosint, routerosinv, routerosperf) and
  the bundled librouteros / chainmap libraries; this TA is now syslog-only
- Added RouterOS v7 firewall event support across the forward, srcnat (masquerade),
  and dstnat chains, capturing connection_state, NAT translation, packet length
  (bytes), TCP flags, and ICMP type/code
- Added RouterOS v7 logout event support (eventtype routeros_auth_logout)
- Tightened CIM Network_Traffic compliance: transport is now lowercased per spec
- Fixed leading-space bug in routeros_auth.csv and routeros_message_type.csv
  lookups that was producing " success" / " Query" instead of "success" / "Query"
- Added FIELDALIAS host AS dest so authentication events populate the CIM
  Authentication dest field
